# manga_metadata_retrieval
Retrieve metadata and cover art for manga and webtoons.
It also can connect to Komga server for updating metadata.

# Avalible providers:
 - MangaDex.
 - MangaUpdates.
 
# What it updates:
 - Main title.
 - Alt titles.
 - Tags.
 - Summary.
 - Cover Art.

# How to use?
You can look at jupter notebooks for examples.
